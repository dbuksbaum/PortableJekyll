    class TZInfo::ZoneinfoDirectoryNotFound

    A ZoneinfoDirectoryNotFound exception is raised if no valid zoneinfo directory could be found when checking the paths listed in TZInfo::ZoneinfoDataSource.search_path. A valid zoneinfo directory is one that contains timezone files, a country code index file named iso3166.tab and a timezone index file named zone1970.tab or zone.tab.

    [Validate]
